# MYNAH Design

Design direction:
- minimal
- clean
- black on white
- low-noise
- functional before decorative

Principles:
- avoid visual clutter
- prefer clear hierarchy over ornamental styling
- keep interfaces calm and professional
- make system state easy to scan
- use whitespace intentionally
- keep diagrams and docs readable first

The design language should feel:
- precise
- modern
- restrained
- technical

Default bias:
- white background
- black text
- subtle gray structure
- strong typography
- minimal accent usage only when it improves clarity
